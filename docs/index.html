OmniBar

Enemy cooldown tracking for Project Ascension

Download Latest Release

Installation

  1. Download OmniBar.zip from the release page.
  2. Extract the zip file.
  3. Move OmniBar and OmniBar_Options to Where/Interface/AddOns/.

Features

  • Tracks cooldowns on Nameplates
  • Customizable standalone bar
  • Native Ascension API support
