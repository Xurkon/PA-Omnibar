# Omnibar

OmniBar tracks enemy cooldowns. Backported to 3.3.5a for Project Ascension.

## Components
- **OmniBar**: The main addon.
- **OmniBar_Options**: Configuration interface.

## Installation
1. Download the latest release.
2. Extract the `OmniBar` and `OmniBar_Options` folders to your `Interface\AddOns` directory.
3. Restart WoW.
